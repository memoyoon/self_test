# 엠패스테스트 (Empath Test)

🧠 **공감 능력치 테스트** — 사용자의 질문 응답을 바탕으로 공감력(Emotional Empathy)을 시각적으로 보여주는 퍼스널리티 테스트 웹 앱입니다.

## 🛠 기술 스택

- React 19
- Vite
- Tailwind CSS
- ESLint
- React Router v7
- Lucide Icons
- html2canvas

## 🧩 폴더 구조

src/
├── assets/ # 이미지, 폰트 등 정적 리소스
├── components/ # UI 컴포넌트
├── data/ # 질문/결과 데이터
├── pages/ # 라우팅되는 페이지 컴포넌트
├── tools/ # 유틸리티 함수들
├── App.jsx
└── main.jsx
